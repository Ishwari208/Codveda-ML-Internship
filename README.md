# Codveda Machine Learning Internship Tasks

This repository contains the Machine Learning tasks completed as part of the Codveda Technologies Internship Program.

## Tasks Completed

### 1. Logistic Regression for Binary Classification
- Dataset Used: Churn Dataset
- Implemented Logistic Regression using Scikit-learn
- Performed data preprocessing and label encoding
- Evaluated the model using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report
  - ROC Curve and AUC Score

### 2. Decision Tree Classification
- Dataset Used: Iris Dataset
- Implemented Decision Tree Classifier
- Performed label encoding
- Visualized the Decision Tree
- Evaluated the model using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab
